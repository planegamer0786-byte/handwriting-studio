Handwriting Studio

Convert typed text into realistic handwritten notes on MJCET Assignment Sheets, ruled paper, grid paper, and more.

v2.1 · React + FastAPI · Open Source

MJCET Assignment Sheet Template

Rendered output: "Network Security" notes on MJCET Assignment / Tutorial Sheet

5 Paper Templates

Plain, Ruled, Grid, MJCET Answer Sheet, and MJCET Assignment Sheet with accurate headers.

Realistic Handwriting

Per-glyph noise transforms with baseline jitter, pressure variance, rotation, and speed blur.

Scanner Effect

Paper grain, elliptical vignette, warmth shift. Cached per canvas size: ~22ms hot path.

Export PNG/PDF

Multi-page export with Celery background processing and graceful thread fallback.

Quick Start

1. Clone & Setup

# Clone the repo
git clone https://github.com/planegamer0786-byte/handwriting-studio.git
cd handwriting-studio

# Download Caveat fonts to backend/fonts/
# Get them from https://fonts.google.com/specimen/Caveat

2. Backend (Python)

cd backend
python -m venv venv
# Windows: venv\Scripts\activate
# Linux/Mac: source venv/bin/activate
pip install -r requirements.txt
python -m uvicorn main:app --host 0.0.0.0 --port 8000

3. Frontend (React)

cd frontend
npm install
npm run dev
# Open http://localhost:5173

4. Build for Production

cd frontend
npm run build
# Static files in frontend/dist/
